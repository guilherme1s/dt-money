# DT Money

Aplicação web para gerenciamento financeiro, permitindo o registro e categorização de transações (entradas e saídas) com atualização dinâmica da interface.

## Funcionalidades

- Cadastro de transações (entrada/saída)
- Validação de formulários com React Hook Form + Zod
- Filtro de transações por descrição
- Atualização automática da lista após criação
- Cálculo automático de resumo financeiro (entradas, saídas e total)
- Interface acessível utilizando componentes do Radix UI

## Decisões Técnicas
- **Context API** e **Context Selector** para gerenciamento de estado global
- **Axios** para comunicação com API mock (JSON Server)
- **React Hook Form + Zod** para validação declarativa e tipagem segura
- **Styled Components** para estilização baseada em componentes
- Organização do projeto por responsabilidade (components, contexts, services)

## Tecnologias

- React
- TypeScript
- Styled Components
- React Hook Form
- Zod
- Context API
- Axios
- Radix UI
- JSON Server
